Graph Visualizer

1.What is Graph visualizer?

This is a software to produce graph from some data.In normal life we see various data.But if we watch is in Graphical format then it will easy to understand for us.

2.How to use Graph Visualizer?

There is several button present in this software.And one selector.First of all you have to select which type of graph you want to see.That are:

  • Line plot
  • Bar plot
  • Scatter plot
  • Histogram plot
  • Pie plot
  • Stack plot
Now the next button name help,that is for help in short purpose.Then there is a button for taking screen shot. Press the button then it will keep the screenshot in E drive.Next button is for tutorial.where you can find detail description how to use this software.Next button is clr for clearing the whole screen.Last button is view that will produce graph as per your code.
First if you want to see line plot then first write the code in this way: x [a,b,c,......] y [a,b,c,....].You have to maintain proper space.This space is equal to 1 spacebar in your keyboard. Otherwise is your don't maintain proper syntax it will give you error.
Same type of thing you have to do for Histogram plot,scatter plot and bar plot.
There is some oher syntax for pie plot and stack plot.For pie plot you have to write as follows: x [a,b,...] .keep it mind a+b+...=100.Otherwise it will give an error.
For Stack plot you have to write as follow: x [a,b,c...] y [a,b,c..] stack. or x [a,b,c...] y [a,b,c..] z [a,b,c...] m [a,b,c..] stack or x [a,b,c...] y [a,b,c..] z [a,b,c,..] stack or x [a,b,c...] y [a,b,c..] z [a,b,c...] m [a,b,c..] m [a,b,c...] stack.You can plot upto 4 for now.
Note: You can give any title in the place of x,y,z etc and inside bracket you have to give only integer.
